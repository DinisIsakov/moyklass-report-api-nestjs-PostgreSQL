# Инструкции по использованию





**Установка зависимостей:**

```bash
$ npm install
```


**Настройка базы данных:**

type: 'postgres',

host: 'localhost',

port: 5432,

username: 'postgres',

password: 'postgres',

database: 'postgres',

**Запуск приложения:**

```bash
$ npm run start
```
